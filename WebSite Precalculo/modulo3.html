División de polinomio

La división de polinomios es un proceso similar a la división de números. Se utiliza para dividir un polinomio (dividendo) por otro polinomio (divisor) y obtener un cociente y, en algunos casos, un residuo.

Ejemplo Supongamos que queremos dividir 2x3+3x2−5x+62x^3 + 3x^2 - 5x + 62x3+3x2−5x+6 entre x−1x - 1x−1:

1. Divide: 2x3÷x=2x22x^3 \div x = 2x^22x3÷x=2x2.

2. Multiplica: 2x2(x−1)=2x3−2x22x^2(x - 1) = 2x^3 - 2x^22x2(x−1)=2x3−2x2.

3. Resta: (2x3+3x2)−(2x3−2x2)=5x2(2x^3 + 3x^2) - (2x^3 - 2x^2) = 5x^2(2x3+3x2)−(2x3−2x2)=5x2.

4. Baja el siguiente término: 5x2−5x=5x2−5x5x^2 - 5x = 5x^2 - 5x5x2−5x=5x2−5x.

5. Divide: 5x2÷x=5x5x^2 \div x = 5x5x2÷x=5x.

6. Multiplica: 5x(x−1)=5x2−5x5x(x - 1) = 5x^2 - 5x5x(x−1)=5x2−5x.

7. Resta: (5x2−5x)−(5x2−5x)=0(5x^2 - 5x) - (5x^2 - 5x) = 0(5x2−5x)−(5x2−5x)=0.

8. Baja el siguiente término: 0+6=60 + 6 = 60+6=6.

9. Divide: 6÷(x−1)6 \div (x - 1)6÷(x−1) es el residuo.

Ceros reales de Funciones Polinomiales

Los Ceros reales de una función polinómica son los valores de xxx para los cuales el polinomio se iguala a cero. Encontrar estos ceros es fundamental para analizar el comportamiento de la función.

Ejemplo Consideremos el polinomio P(x)=x3−6x2+11x−6P(x) = x^3 - 6x^2 + 11x - 6P(x)=x3−6x2+11x−6.

1. Intentar la factorización: Podemos probar con valores enteros (como 1,2,31, 2, 31,2,3).

2. Evaluamos: • P (1) =1−6+11−6=0P (1) = 1 - 6 + 11 - 6 = 0P (1) =1−6+11−6=0 (cero). • P (2) =8−24+22−6=0P (2) = 8 - 24 + 22 - 6 = 0P (2) =8−24+22−6=0 (cero). • P (3) =27−54+33−6=0P(3) = 27 - 54 + 33 - 6 = 0P(3)=27−54+33−6=0 (cero).

3. Factorización: Sabemos que P(x)=(x−1) (x−2) (x−3) P(x) = (x - 1) (x - 2) (x - 3) P(x)=(x−1) (x−2) (x−3).

Por lo tanto, los ceros reales son x=1x = 1x=1, x=2x = 2x=2 y x=3x = 3x=3.

Ceros Complejos y el teorema fundamental del Algebra

El Teorema Fundamental del Álgebra establece que todo polinomio de grado nnn (donde n>0n > 0n>0) con coeficientes complejos tiene exactamente nnn raíces complejas, contando multiplicidades.

Ejemplo

Consideremos el polinomio P(x)=x3+2x2+5x+6P(x) = x^3 + 2x^2 + 5x + 6P(x)=x3+2x2+5x+6. Para encontrar sus raíces, podríamos usar métodos como la factorización, la regla de signos de Descartes, o métodos numéricos. Si encontramos que el polinomio tiene una raíz compleja, como 1+i1 + i1+i, por el teorema de conjugación de los números complejos, también debe tener su conjugado, 1−i1 - i1−i como raíz.
